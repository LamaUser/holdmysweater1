Overview

This is a simple AI text generation API endpoint that you can use in your applications. It's designed for future monetization with usage-based pricing.

API Endpoint

POST /api/v1/generate

Request Body

{ "prompt": "Your text prompt here", "model": "mistralai/Mistral-7B-Instruct-v0.2", // optional "maxTokens": 200, // optional, default: 200 "temperature": 0.7 // optional, default: 0.7 }

Response

{ "success": true, "data": { "text": "Generated text response", "model": "mistralai/Mistral-7B-Instruct-v0.2", "tokens": 150, "createdAt": "2024-01-01T00:00:00.000Z" } }

Usage Examples

cURL

curl -X POST https://your-domain.com/api/v1/generate \ -H "Content-Type: application/json" \ -d '{"prompt": "Write a short story about a robot"}'

JavaScript (Fetch)

const response = await fetch('https://your-domain.com/api/v1/generate', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ prompt: 'Write a short story about a robot', maxTokens: 300 }) }); const data = await response.json(); console.log(data.data.text);

Python (requests)

import requests response = requests.post( 'https://your-domain.com/api/v1/generate', json={ 'prompt': 'Write a short story about a robot', 'maxTokens': 300 } ) data = response.json() print(data['data']['text'])

Monetization Strategy

  • Free Tier: Limited requests per day (e.g., 100 requests)
  • Pro Tier: Higher limits, faster responses
  • Enterprise: Custom pricing, dedicated support
  • API Keys: Implement authentication for usage tracking
  • Rate Limiting: Prevent abuse and manage costs
